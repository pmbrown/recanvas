recanvas

A color-scheme friendly reset CSS file with standardized typography, forms, buttons and tables, recanvas is a lightweight 2.53kb unminified style sheet written as a basic foundation for web design. Shared under a MIT license and available for download from Github the examples below demonstrate the styles provided.
